## Reponsive Shoe Web Page

## Overview
- This project is a responsive product landing page inspired by modern shoe brands such as Puma and Anta. It showcases footwear  using an interactive, swipe-based layout to create an engaging browsing experience for users.

- The project was built using a tutorial as a foundation, with customization and implementation done independently to better understand responsive layouts and interactive UI components.

## Features 
- Swipeable product carousel using Swiper.js
- Hover effects for improved user interaction  
- Mobile-friendly navigation with toggle and close functionality
- Fully responsive layout for desktop and mobile devices

## How it works 
- Users can hover over a shoe and swipe left or right to preview different products. On smaller screens, navigation adapts to a toggle-based menu to improve usability on mobile devices.. 

## What I learned 
- How to implement and customize Swiper.js for interactive product displays.
- Building responsive layouts that adapt to different screen sizes.
- Creating mobile navigation with toggle and close functionality
- Improving user experience through hover and motion-based interactions

## Technology Used
- HTML
- CSS
- Javascript
- Swiper.js
